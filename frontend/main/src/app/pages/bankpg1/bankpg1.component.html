BANKER'S ALGORITHM

The banker's algorithm is a deadlock avoidance algorithm used in operating systems. It is used to avoid the occurrence of deadlock by ensuring that the resources are allocated to the processes in a safe manner.The algorithm works by using a set of matrices to represent the available resources. The algorithm uses these matrices to determine if a particular resource allocation request will leave the system in a safe state, that is, a state in which no deadlock can occur.

  • Advantages

    • 1. It contains various resources that meet the requirements of each process.
    2. Each process should provide information to the operating system for upcoming resource requests, the number of resources, and how long the resources will be held.

  • Disadvantages

    • 1. It requires a fixed number of processes, and no additional processes can be started in the system while executing the process.
    2. The algorithm does no longer allows the processes to exchange its maximum needs while processing its tasks.
